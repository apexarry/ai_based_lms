# AI Based LMS

An AI-based Library Management System designed to help researchers search, explore, and interact with research papers and books using Artificial Intelligence.

## 🚀 Planned Features

- User Authentication
- Digital Library Management
- PDF Upload & Storage
- OCR for Scanned Documents
- AI Chatbot (RAG) for Question Answering
- AI-Based Recommendation Engine
- Semantic Search
- Reading History

## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL

### AI
- Groq
- Antigravity
- ChromaDB
- PaddleOCR
- Sentence Transformers
